## コーディングルール

### プラグイン拡張用のhookについて
後からプラグインで変更できるようなメソッド全てにhookを設定する。

各クラスファイルのそれぞれのメソッド内の最後にhookを定義する場合のhook名を統一する。

hook名は全て小文字で
```
クラス名_メソッド名
```
とし、可読性を高めるために変数や定数は使用禁止。

メソッドの最後に
```
return apply_filters('クラス名_メソッド名', $変数);
```
とする場合に適用

デフォルトではない特殊なhookを定義する場合はこのルールは適応外
